Vagrant在windows下加载的文件全部是777，但是在Mac 下会是644
这样会导致文件权限问题。

解决方法如下：

```
    config.vm.synced_folder "/Users/vincent/www","/vagrant",:mount_options => ["dmode=777","fmode=777"]
```
